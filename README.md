Privacy Policy

Alaska State | Official

Last Updated: May 5, 2026

---

What This Is

This Privacy Policy explains how Alaska State | Official (the "Bot") handles information when you use it in Discord servers. This Bot is part of the Alaska State roleplay game — a fictional community with no connection to real-world government agencies.

---

What We Collect

The Bot only collects what it needs to function. Depending on your server's settings, this may include:

Data	Why We Need It	
Discord User ID	To identify users for moderation and logging	
Username and avatar	To display in logs and moderation records	
Message content	Only if message logging is enabled by server admins	
Voice channel activity	Only if voice logging is enabled by server admins	
Server roles and permissions	To enforce role-based commands and protections	
Moderation actions	To keep records of kicks, bans, warnings, etc.	

We do not collect real names, addresses, emails, phone numbers, payment info, or anything outside of Discord.

---

How We Use It

- Server operations: Logging, moderation, and security features
- Roleplay management: Supporting the Alaska State roleplay game
- Bug fixes and improvements: Making the Bot work better

We don't use your data for advertising, profiling, or anything unrelated to the Bot's core functions.

---

Where Data Lives

- Data is processed and stored within the Bot's hosting infrastructure
- Logs are typically kept based on server-configured retention settings
- We don't share data with third parties, sell it, or move it between servers

Server administrators control what gets logged and how long it's kept.

---

Who Can See It

- Bot operators: Developers maintaining and troubleshooting the Bot
- Server admins/moderators: People your server owner has given access to logs
- Discord: If legally required or for platform safety investigations

Regular server members can't see logs unless admins explicitly grant access.

---

Your Rights

Since this is a Discord bot, your main controls are through Discord and your server:

- Leave the server: Stops the Bot from collecting new data about you
- Request deletion: Ask the server admin to remove your logged data
- Contact us: Reach out if you have concerns about how data is handled

We can't delete data from Discord itself — that's between you and Discord.

---

Data Retention

We keep data only as long as needed for the Bot to function or as configured by server administrators. Old logs may be automatically deleted based on server settings. If the Bot is removed from a server, associated configuration data is typically purged.

---

Security

We take reasonable steps to protect data, but no system is perfect. The Bot operates within Discord's security framework. Don't share sensitive personal information through Discord or this Bot — it's not designed for that.

---

Children's Privacy

The Bot follows Discord's age requirements. If you're under 13 (or your country's digital consent age), you shouldn't be on Discord or using this Bot. If we learn we've collected data from someone underage, we'll delete it.

---

Changes to This Policy

We may update this Privacy Policy as the Bot evolves. Significant changes will be announced in our support server. Continued use after updates means you accept the revised policy.
